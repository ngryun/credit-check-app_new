## 교육과정 편성표 업로드 양식 (확정)

### 파일 형식
- **xlsx 파일** (Excel)
- 셀 병합이 포함되어 있음
- 샘플 파일: `../2024년_입학생편성표_2026-03-19.xlsx`

### 컬럼 구조
| 열 | 내용 | 비고 |
|----|------|------|
| A | 과목구분 | "학교지정" 또는 학생선택 그룹명. **셀 병합**되어 있음 |
| B | 교과(군) | 국어, 수학, 영어 등. **같은 값 연속 시 셀 병합** |
| C | 과목유형 | 일반선택 / 진로선택 |
| D | 과목명 | |
| E | 비고 | 현재 불필요 |
| F | 총 학점 합계 | 전체 학기 합산값, 현재 불필요 |
| G | 1학년 1학기 학점 | 0이면 해당 학기 미개설 |
| H | 1학년 2학기 학점 | |
| I | 2학년 1학기 학점 | |
| J | 2학년 2학기 학점 | |
| K | 3학년 1학기 학점 | |
| L | 3학년 2학기 학점 | |

### 특수 행 처리
- **과목구분 변경 행**: B, C, D열이 공란
  - 해당 선택그룹의 과목선택 수 제약 정보가 포함됨
  - 예: `학기 별 과목수 최소선택 : 5 ~ 최대선택 : 5`
  - 이 행은 **데이터 행이 아님** → 메타정보로 파싱 필요 (시뮬레이션 기능에서 사용)
- **셀 병합 처리**: A열(과목구분), B열(교과군)은 병합셀 → 빈 셀은 위의 값을 상속

### 파서 수정 방향 (`src/lib/curriculum-catalog.ts`)
1. xlsx 파일의 병합셀을 해제하여 읽기 (또는 빈 셀에 위 값 채우기)
2. B/C/D열이 공란인 행 → 선택그룹 메타정보 행으로 처리
3. G~L열에서 0이 아닌 값이 있는 학기를 해당 과목의 개설 학기로 매핑
4. 가로 피벗 → 세로 구조로 변환하여 기존 로직과 호환

## 신규 기능: 미래 과목선택 시뮬레이션
대시보드에서 아직 수강하지 않은 미래 학기의 과목을 직접 선택해보며:
- **이수기준 충족 여부** 실시간 점검
- **과목 위계(선수과목)** 충족 여부 검증
- **선택그룹별 최소/최대 선택 수** 제약 조건 검증
  - 메타정보 행에서 파싱한 `최소선택`, `최대선택` 값 활용
- 선택 결과에 따른 졸업요건 달성률 시뮬레이션

→ 기존 `future-selection.ts`, `prerequisite-check.ts` 로직 확장 필요
